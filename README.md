# WeiKnight's OC Formula Book (WIP)

## 📖 Content Introduction

This is a comprehensive set of setting materials written for WeiKnight's original characters (OCs) and their associated worldview, which is mostly written in CHINESE, including:

### 🌍 Worldview Section
- Basic settings (timeline/geography/civilization/special laws)
- Organizational structure of forces
- Important historical events
- Cultural customs and technological level
- Other supplementary settings

### 🎭 Character Profiles
- Full profiles of core OCs
- Picture books/illustrations of related characters
- Character relationship network
- Explanations of clothing/prop design drafts

---

## 🛠 Usage Instructions

1. **File Structure**:
    - `settingTEX.tex` is the main file.
    - `settingTEX.pdf` is the visual file.

2. **File Formats**:
   - Main language: `TEX`
   - Illustrations: PNG/JPG
